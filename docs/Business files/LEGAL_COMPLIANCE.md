## Data Privacy & Protection

### Personal Data Protection Act (PDPA) - India

**Applicability:**
- Applies to processing of personal data in India
- Includes health data
- Applies to businesses processing personal data

**Key Requirements:**

1. **Consent**
   - Obtain explicit consent
   - Clear consent language
   - Easy to withdraw
   - Document consent

2. **Purpose Limitation**
   - Collect only necessary data
   - Use for stated purpose only
   - Don't use for other purposes

3. **Data Minimization**
   - Collect minimum required data
   - Don't collect unnecessary data
   - Delete when no longer needed

4. **Data Security**
   - Encryption (at rest & in transit)
   - Access controls
   - Security measures
   - Regular audits

5. **Data Subject Rights**
   - Right to access
   - Right to correction
   - Right to deletion
   - Right to portability

**Compliance Actions:**
- [ ] Privacy policy
- [ ] Consent mechanisms
- [ ] Data security measures
- [ ] Data subject rights process
- [ ] Regular audits

---

### GDPR Compliance (If EU Customers)

**Applicability:**
- If processing EU residents' data
- If offering services in EU
- Stricter than PDPA

**Key Requirements:**

1. **Lawful Basis**
   - Consent
   - Contract
   - Legal obligation
   - Legitimate interest

2. **Data Protection Officer (DPO)**
   - May be required
   - For large-scale processing
   - Healthcare data

3. **Data Breach Notification**
   - Report within 72 hours
   - Notify data subjects
   - Document breaches

4. **Privacy by Design**
   - Built-in privacy
   - Default privacy settings
   - Data minimization

**Compliance Actions:**
- [ ] GDPR assessment
- [ ] Privacy policy (GDPR-compliant)
- [ ] Data processing agreements
- [ ] Breach notification process
- [ ] DPO (if required)

## Business Compliance

### Company Registration

**Entity Type Options:**

1. **Private Limited Company**
   - Most common for startups
   - Limited liability
   - Professional image
   - Easier funding

2. **LLP (Limited Liability Partnership)**
   - Partnership structure
   - Limited liability
   - Flexible
   - Less formal

3. **Sole Proprietorship**
   - Simplest
   - Unlimited liability
   - Less professional
   - Harder to scale
